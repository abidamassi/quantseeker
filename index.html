ALGORITHMIC TRADING: RETURN DI ATAS OBLIGASI, TANPA RISIKO MODAL.

Sistem kami memanfaatkan kombinasi AI, model matematika dan statistika, serta indikator teknikal yang secara sistematis mencari profit yang maksimal.

HANYA SISTEM YANG BISA MENGALAHKAN PASAR

RISIKO TANPA SISTEM

  • Keputusan yang dipengaruhi emosi rentan merugi.
  • Analisis terbatas pada indikator umum dan manual.
  • Trading manual rentan "diguyur" oleh bandar.
  • Sulit beradaptasi dengan perubahan pasar yang cepat.
  • Potensi kehilangan modal investasi.
  • Data backtesting seringkali tidak valid atau overfitting.

POTENSI MEMAKAI SISTEM

  • 100% keputusan berbasis data, bebas emosi manusia.
  • Algoritma kompleks: matematis, statistik, indikator teknikal, dan machine learning.
  • Potensi return Anda konsisten 10% per tahun di atas benchmark obligasi.
  • Tanpa risiko bagi Anda. Kami menanggung semua risiko dengan jaminan cash 1:1 dengan modal Anda.
  • Alat backtesting dan validasi strategi yang akurat.

APAKAH BENAR TANPA RISIKO?

Tidak ada investasi yang tidak berisiko. Semua tentu memiliki risiko. Namun, dalam model ini, Quant Seeker yang menanggung seluruh risiko tersebut. Bagi Anda sebagai investor, hasilnya sesuai kontrak: modal aman dan return 10% konstan annually.

LOGIKA DI BALIK ALGORITMA

Projek kami dibangun di atas sistem dan kode yang robust. Berikut adalah contoh cuplikan kodenya (versi sederhana). 


class ExampleStrategy(Strategy):
    @property
    def slow_sma(self):
        return ta.sma(self.candles, 200)

    @property
    def fast_sma(self):
        return ta.sma(self.candles, 50)

    def should_long(self) -> bool:
        # Fast SMA above Slow SMA
        return self.fast_sma > self.slow_sma

    def should_short(self) -> bool:
        # Fast SMA below Slow SMA
        return self.fast_sma < self.slow_sma

    def should_cancel_entry(self) -> bool:
        return False

    def go_long(self):
        # Open long position and use entire balance to buy
        qty = utils.size_to_qty(self.balance, self.price, fee_rate=self.fee_rate)

        self.buy = qty, self.price

    def go_short(self):
        # Open short position and use entire balance to sell
        qty = utils.size_to_qty(self.balance, self.price, fee_rate=self.fee_rate)

        self.sell = qty, self.price

    def update_position(self):
        # If there exist long position, but the signal shows Death Cross, then close the position, and vice versa.
        if self.is_long and self.fast_sma < self.slow_sma:
            self.liquidate()
    
        if self.is_short and self.fast_sma > self.slow_sma:
            self.liquidate()

PERTANYAAN UMUM

Apa itu Quant Seeker?

Quant Seeker adalah sebuah private investment fund (dana investasi privat) yang sepenuhnya dikelola oleh algoritma.

Berapa imbal hasil yang ditawarkan?

Kami menawarkan target imbal hasil tetap sebesar 10% per tahun. Angka ini dirancang untuk lebih tinggi dari instrumen konvensional seperti deposito atau obligasi.

Jika sistem bisa untung besar, kenapa saya hanya dapat 10%?

Karena Anda memilih produk dengan imbal hasil tetap dan proteksi modal. Selisih profit digunakan oleh kami sebagai "dana cadangan" untuk menanggung seluruh risiko pasar dan menjamin keamanan modal Anda.

Apakah investasi ini benar-benar tanpa risiko?

Semua investasi memiliki risiko. Namun, dalam model ini, Quant Seeker yang menyerap dan mengelola seluruh risiko pasar tersebut. Bagi Anda sebagai investor, hasilnya sesuai perjanjian: modal aman dan imbal hasil 10%.

Berapa dana minimal dan maksimal untuk berinvestasi?

Minimal investasi adalah Rp5 juta dan maksimal Rp20 juta per investor. Batasan ini kami terapkan untuk menjaga efektivitas protokol proteksi modal kami.

Apa yang membuat modal saya aman di sini?

Karena kami menerapkan sistem dana cadangan 1:1. Setiap dana investasi Anda, kami mencadangkan dana tunai kami sejumlah yang sama sebagai penjamin. Inilah alasan utama mengapa slot investasi kami terbatas.

Di instrumen mana saja dana saya diinvestasikan?

Sistem kami bertransaksi di berbagai instrumen pasar global secara otomatis, termasuk saham (equities), futures, options, dan cryptocurrencies.

Apa dasar dan indikator dari sistem trading Quant Seeker?

Sistem kami dibangun di atas tiga pilar: model matematis-statistik, ribuan indikator teknikal, dan machine learning dengan integrasi AI.

Bisakah saya mendapat imbal hasil lebih dari 10%?

Ya, bisa. Kami memiliki skema profit sharing untuk imbal hasil yang lebih tinggi. Dalam skema tersebut, Anda ikut berpartisipasi dalam risiko investasi. Hubungi kami untuk diskusi lebih lanjut.

Jika sistemnya sangat profit, mengapa tidak pinjam dari bank saja?

To be frank, karena mahal. Analisis kami, menggunakan metrik finansial seperti Internal Rate of Return (IRR), menunjukkan bahwa total biaya efektif pinjaman bank jauh lebih tinggi dibandingkan skema bagi hasil 10%.

Bagaimana jika sistem gagal atau merugi?

Protokol proteksi modal dan dana cadangan 1:1 kami dirancang untuk situasi ini. Jika target tidak tercapai atau sistem merugi, modal awal Anda akan dikembalikan utuh sesuai perjanjian.
